과제2
scanf() = input()같은 애

scanf("%d", &a)
하면 a에 입력되는거
다음에
printf("%d",a); 가 있으면
입력값 적은게 a고 그 a가 프린트됨

scanf에서 변수 앞에는 &가 붙는데
&는 변수의 주소를 가리킴.(주소=저 변수가 위치하는 곳으로 가)

(주소: https://m.blog.naver.com/tksop59/221881055008)

Week2

int x; x=10; -> x=10
int *p; -> p=주소, p=&x -> *p=10

<아니 그러면 *는 뭔 역할이 정확히 뭐야>
ㄴ 자자 x라는 내용.
ㄴ &x=쪽수=100
ㄴ 그럼 p가 뭐야
ㄴ *p=내용




int s[3];
-> s[0], s[1], s[2]
=   *s  *s+1  *s+2 (주소)

포인터 쓰는 이유
ex. 어떤 사전에서 100쪽 내용을 공유하고 싶어

그니까 100쪽이라는 것만 알려주는거지
